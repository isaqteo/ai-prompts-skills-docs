# Meu Repositório de IA

Repositório pessoal para versionar **Skills**, **Prompts** e **Documentação** usados em fluxos de trabalho com IA (Claude, Notion AI, etc.).

## Estrutura

- `skills/` — Skills no formato SKILL.md, uma pasta por skill.
- `prompts/` — Prompts organizados por ferramenta de uso.
- `docs/` — Manuais de estilo e documentação de processo (espelha a base "Documentação" do Notion).

## Relação com o Notion

Este repositório funciona como backup versionado do conteúdo que vive ativamente no Notion:

| Pasta local        | Base no Notion       |
|---------------------|-----------------------|
| `docs/`             | Documentação          |
| `prompts/notion-ia/`| Prompts               |
| `skills/`           | (não existe no Notion — específico do Claude) |

## Índice de Skills

- `prompt-master` — Gera prompts otimizados para qualquer ferramenta de IA.
- `organizador-de-prompts` — Organiza e padroniza a biblioteca de prompts no Notion.
