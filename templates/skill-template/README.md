# templates/skill-template

Cópia de referência do mesmo modelo usado em `skills/_template/`. A diferença
de propósito entre as duas pastas:

- `skills/_template/` vive dentro de `skills/` e é o modelo "ao vivo" — a
  primeira coisa que se vê ao abrir a pasta de skills, pronta para ser
  copiada e renomeada.
- `templates/skill-template/` vive junto dos outros templates do repositório
  (`prompt-template.md`, `context-template.md`, `persona-template.md`), para
  quem está navegando por `templates/` em busca de "qual modelo eu uso para
  criar X" sem precisar saber de antemão que skills têm seu próprio template
  dentro de `skills/`.

Os dois arquivos `SKILL.md` são mantidos idênticos de propósito. Ao editar um,
edite o outro.
